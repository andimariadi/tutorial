Cara install aplikasi

Jika database masih kosong dan tidak memeliki username dengan Admin, maka buat terlebih dahulu user dengan level admin di localhost/ujian/install.php. Maka akan tampil gambar seperti dibawah ini :

Masukan nama username dan password yang akan dikehendaki, misalnya :

username : admin

password : admin

Jika sudah maka user dengan admin sudah dibuat, langsung masuk ke http://localhost/ujian untuk login ke system CAT.
